# Code of Conduct

This project follows the **Contributor Covenant v2.1** as its code of conduct.

The full text is published at:
**https://www.contributor-covenant.org/version/2/1/code_of_conduct/**

By participating in this project — opening issues, submitting pull requests, commenting in discussions, or any other interaction in project channels — you agree to abide by its terms.

## Enforcement

Reports of conduct that violates the Covenant can be sent to **conduct@querylabs.ai**.

All reports will be reviewed and investigated promptly and fairly. Project maintainers are obligated to respect the privacy and security of the reporter of any incident.

## Scope

This Code of Conduct applies in all project spaces — the GitHub repository (issues, discussions, pull requests, code comments), any official communication channels, and when an individual represents the project in public spaces.
